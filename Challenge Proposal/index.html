Overview

Purpose

The application will use the Pokémon API to create a team of Pokémon creatures. The user will have a form that enables him to select specific Pokémon and store them together as a team. It will show the image of the Pokémon and some of their characteristics. The user will then be able to add, remove and update his team in local storage. The purpose of this application is to apply the concepts I learned in this class.

Audience

The audience are people that like Pokémon and want to make a team of them together. They can compare the creatures and choose which ones they want on their team.

Data Sources

  1. Local Storage: I'll use the localStorage to build the team of Pokémons. It will be possible to add and remove the Pokémons from the localStorage.
  2. Local JSON file: Save the team of the localStorage in the file.
  3. Form: the form will allow the user to search the Pokémon he wants
  4. External API: I will use the Pokémon API

Initial Module list

  • index.html: HTML Code
  • style.css: CSS Code
  • main.js: the main JavaScript file
  • I'll try to apply the MVC pattern so initially the modules files I think will be:
    • PokemonView.js
    • PokemonRequest.js
    • PokemonModel.js

Wireframes

In the wireframe below we can see there will be 3 sections to this page:

  1. At the top we have a section that is the navigation bar. With the links for the Home (which is this page) and the Team pages.
  2. After, we have a section that will have the search bar, and the result appears in the bottom. The image of the pokemon is displayed followed by its characteristics and the add to the team buttom.
  3. Lastly, we have a section with a list of all the Pokemons. This is in case the user doesn't know the names.

In the wireframe below we can see there will be 2 sections to the team page:

  1. At the top we have a section that is the navigation bar. With the links for the Home and the Team (which is this page) pages.
  2. After we have a section that will show the team.
  3. The Pokemons that were added to the team will be showing with their pictures and names. They will have a button that will allow the user to remove it from the team.
